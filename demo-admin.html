Getting Started

1

Login as Admin

Access the admin panel by logging in with your admin credentials at /login.html

2

Choose Your Task

Navigate to the visual admin panel at /admin-visual and select what you want to edit from the dashboard

3

Edit Visually

Click "Edit Page" to open any page in visual edit mode. Click on elements to modify them directly

4

Manage Trips

Use the Trips tab to add new trips, edit existing ones, or delete trips with the + and - buttons

5

Save & Publish

All changes are automatically saved. Click "Save Changes" to publish your modifications
